CSS3 Shadows

Overview

Use drop shadows on elements and text in place of images.

Demo

text-shadow: 2px 2px 1px #374252;

box-shadow: 1px 2px 4px rgba(0, 0, 0, .5);
box-shadow: 0 15px 20px rgba(0, 0, 0, 0.3);
box-shadow: 5px 5px 0 #374252;

Code

  • html
  • CSS

<h1 class="text-shadow">text-shadow: 2px 2px 1px #374252;</h1>
                    
<div class="demo box-shadow-1 pull-left">box-shadow: 1px 2px 4px rgba(0, 0, 0, .5);</div>
<div class="demo box-shadow-2">box-shadow: 0 15px 20px rgba(0, 0, 0, 0.3);</div>
<div class="demo box-shadow-3 pull-right">box-shadow: 5px 5px 0 #374252;</div>

.text-shadow {
color: #ccc;
font-family: monospace;
text-shadow: 2px 2px 1px #374252;
}
    
.demo {
background: #fff;
float: left;
padding: 15px;
width: 30%;
}

.box-shadow-1 {
   -moz-box-shadow: 1px 2px 4px rgba(0, 0, 0,0.5);
-webkit-box-shadow: 1px 2px 4px rgba(0, 0, 0, .5);
        box-shadow: 1px 2px 4px rgba(0, 0, 0, .5);
}

.box-shadow-2 {
margin-left: 60px;
   -moz-box-shadow: 0 15px 20px rgba(0, 0, 0, 0.3);
-webkit-box-shadow: 0 15px 20px rgba(0, 0, 0, 0.3);
        box-shadow: 0 15px 20px rgba(0, 0, 0, 0.3);
}

.box-shadow-3 {
border: 1px solid #4C6575;
   -moz-box-shadow: 5px 5px 0 #374252;
-webkit-box-shadow: 5px 5px 0 #374252;
        box-shadow: 5px 5px 0 #374252;
}
